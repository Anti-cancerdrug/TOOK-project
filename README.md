TOOK

툭, 던지듯 나오는 세계관의 시작점

1. 프로젝트 개요

TOOK은 세계관을 중심으로 한 창작 활동을 보다 체계적으로 기록하고
콘텐츠(웹툰·소설)와 자연스럽게 연결하기 위해 기획한 1인 개발 프로젝트입니다.

세계관 설정이 단편적인 메모에 흩어지지 않고,
위키 형태로 정리 → 콘텐츠로 확장 → 공유되는 흐름을 만드는 것을 목표로 합니다.

2. 핵심 기능

회원가입 및 로그인

개인 세계관 생성 및 관리

세계관 기반 위키 (인물, 장소, 설정 등 카드형 구조)

CRUD 기반 위키 편집

기본 이미지 자동 처리 (대표 이미지 미등록 시 대체 이미지 적용)

3. 기술 스택

Backend: PHP

Database: MySQL (PDO)

Frontend: HTML, CSS, JavaScript

Environment: XAMPP (Apache)

Version Control: Git, GitHub

4. 프로젝트 구조
took-project/
 ├─ assets/
 ├─ config/
 │   ├─ db.sample.php
 │   ├─ constants.sample.php
 ├─ includes/
 ├─ member/
 ├─ world/
 └─ index.php


실제 DB 및 환경 설정 파일은 보안상 제외하고
sample 파일을 제공하여 구조와 의도를 명확히 했습니다.

5. 진행 상태

✅ 기본 회원 시스템 구현

✅ 세계관/위키 CRUD 구현

🔄 UI 개선 및 기능 고도화 진행 중

본 프로젝트는 현재도 계속 개발 중이며,
기능 단위로 개선과 리팩터링을 진행하고 있습니다.

6. 기획 의도

이 프로젝트는 단순한 웹 서비스 구현이 아닌,
스토리 작가이자 세계관 창작자로서의 문제의식에서 출발했습니다.

과거의 저처럼
“세계관을 어떻게 정리해야 할지 막막한 사람들”을 위한
도구가 되는 것을 최종 목표로 합니다.

7. 향후 계획
